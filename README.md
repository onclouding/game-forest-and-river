一個基於瀏覽器的單機網頁版生態模擬遊戲。玩家扮演造物主，在有限的網格中平衡水資源與木材，建立一個能夠自我繁衍的森林生態系。

## ✨ 遊戲特色

* **單一檔案，極致輕量**：整個遊戲僅由一個 HTML 檔案組成，無需安裝任何依賴或伺服器。
* **動態生態系統**：
    * 樹木在水源附近生長速度加倍。
    * 成熟的森林具備「自我繁衍」能力，會向周圍擴散種子。
    * 河流會持續滋潤土地並產生水資源。
* **資源管理策略**：
    * **💧 水**：種植樹苗的必需品。
    * **🪵 木材**：開鑿河道與擴張水源的材料。
    * **🏆 生態分**：反映你的森林繁榮程度。
